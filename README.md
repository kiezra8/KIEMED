# KIEMED B2B Medical Supplies

A premium React Vite application for wholesale medical supply distribution, designed with a sleek, Shein-inspired aesthetic.

## Features
- **Shein-inspired UI**: Clean, modern design with premium typography and layouts.
- **Dynamic Hero Slider**: 3-slide animation with smooth transitions.
- **Circular Categories**: Large, home-page visible categories for easy navigation.
- **Hierarchical Browsing**: Drugs -> Classes -> Products flow.
- **Smart Search**: Functional search bar for all medical supplies.
- **WhatsApp Integration**:
  - Direct chat option from the Navbar and Chat page.
  - Cart "Order" button that pre-fills a WhatsApp message with order details.
- **Responsive Design**: Works perfectly on mobile and desktop.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

## Note on PowerShell
If you experience errors running commands in your terminal, it may be due to a system PowerShell issue. You can try running commands in **Command Prompt (CMD)** or **Git Bash** instead.
